CHECKOUT FLOW:
1. User navigates to checkout.html
2. Sees order summary with items and total (Rs. 2,586)
3. Fills in billing information form:
   - Full Name
   - Email Address
   - Phone Number
   - Delivery Address
   - City
   - Postal Code
4. Selects payment method:
   - Cash on Delivery (default)
   - Easypaisa
   - JazzCash
5. Clicks "Confirm Order"
6. Form validation runs
7. Success modal displays with:
   - Generated Order ID
   - Total Amount
   - Selected Payment Method

// ==============================================
// JAVASCRIPT FUNCTIONS
// ==============================================

generateOrderId()
- Generates random 6-digit order ID
- Returns: "#123456" format

getPaymentMethodName(value)
- Converts payment method value to display name
- Supported: 'cod', 'easypaisa', 'jazzcash'

validateCheckoutForm()
- Validates all required fields
- Checks: name, email, phone, address, city, zipcode
- Shows alert if validation fails
- Returns: boolean

showSuccessModal(paymentMethod)
- Opens success modal with order details
- Generates order ID
- Updates modal with payment method
- Adds 'show' class to display modal

closeSuccessModal()
- Closes success modal
- Removes 'show' class

initCheckout()
- Initializes event listeners
- Handles "Confirm Order" button click
- Validates form on submit
- Shows success modal if valid
- Handles modal close button
- Handles outside modal click to close
